The format of this element **SHOULD** be an HTTP URI which points to a related resource. It is **RECOMMENDED** that, where available, the publisher’s DOI for the main resource being described in the RIOXX record also be entered here, in it’s HTTP form – e.g. http://dx.doi.org/10.1006/jmbi.1995.0238

Each related resource **MUST** appear as a separate instance of the field.
